📋 API-Visitas

API de estadísticas de visitas web con geolocalización IP, mapa mundial interactivo y dashboard en tiempo real.

🚀 Inicio rápido

1. Instalar e iniciar

cd APIs/API-Visitas
npm install
npm start

2. Integrar en tu proyecto

Agrega este script en el <head> o <body> de tu HTML:

<script src="http://localhost:3000/track?project=TU_PROYECTO&page=inicio"
        async></script>

O vía fetch:

fetch('http://localhost:3000/track?project=calificaciones&page=home')
  .then(r => r.json())
  .then(data => {
    console.log('Visitante desde:', data.visitor.city, data.visitor.country);
    console.log('Navegador:', data.visitor.browser);
  });

3. Ver dashboard

Abre http://localhost:3000/ en tu navegador.

📡 Endpoints

GET/track?project=NOMBRE&page=URL
Registra una visita. Usar como script o fetch. Retorna datos del visitante (país, ciudad, navegador, SO, dispositivo). Soporta JSONP con &callback=func.
GET/api/stats/overview?project=NOMBRE
Total visitas, visitantes únicos, visitas hoy, únicos hoy, esta semana, top páginas.
GET/api/stats/timeline?project=NOMBRE&days=30
Visitas por día (últimos N días).
GET/api/stats/devices?project=NOMBRE
Desglose por navegador, SO, tipo de dispositivo.
GET/api/stats/locations?project=NOMBRE
Datos geográficos (países + ciudades con coordenadas) para el mapa.
GET/api/stats/recent?project=NOMBRE&limit=20
Últimas visitas con detalle.
GET/api/projects
Lista todos los proyectos registrados con estadísticas.
GET/api/health
Health check del servidor.

⚡ Precisión y limitaciones

Geolocalización: Usa ip-api.com (gratis, 45 req/min, sin key). ~80% precisión de ciudad, ~95% país. Localhost/IPs privadas se marcan como "Local".

Filtro de bots: Detecta crawlers (Googlebot, GPTBot, etc.), herramientas (curl, wget) y tráfico de datacenters (AWS, GCP, Azure, DO, OVH).

Visitantes únicos: Por hash MD5 de IP + User-Agent. No usa cookies.

📊 Dashboard

El dashboard incluye:

  • Mapa mundial Leaflet con marcadores por ciudad
  • Cards de estadísticas (totales, hoy, semana, únicos)
  • Gráfico timeline (visitas + únicos por día)
  • Gráfico de navegadores (doughnut)
  • Top páginas más visitadas
  • Tabla de últimas visitas con detalle
  • Selector de proyectos
  • Auto-refresh cada 30 segundos
